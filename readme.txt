-------------------------------------------------------------------------------
                                  File2img
-------------------------------------------------------------------------------


任意のファイルを画像に変換する:
$ python3 file2img.py <元ファイル> <出力ファイル> <画像の一辺のサイズ>


変換したファイルを元に戻す:
$ convert ファイルパス a.bmp && xxd -c 1 -b -s 138 a.bmp | awk '{printf $2}' | perl -ne 'print pack("B*", $_)'

convertで作成されたbmpファイルは138bit目から画像のデータがはじまります。
参考: https://togetter.com/li/1382211


Tips:
Twitterに画像をあげる際は100x100以上にするとデータが欠損しにくいようです。
